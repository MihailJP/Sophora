==============================================================================
手書きフォント Sophora
==============================================================================

■これは何？

	　これはOpenType手書きフォントです。
	　作者がノートにボールペンで文字を書き、それをスキャンしSVGにトレースして作って
	おります。
	　ほとんど調整をしていない安普請な仕上がりですが、普通の日本語フォントには見られ
	ないグリフも出来る限り収録することを目指しています。

■ライセンスについて

	　ライセンスは、マイナーバージョン番号(例えば1.2.3と表記された場合は2)が偶数か
	奇数かによって異なります。

	▼マイナーバージョンが偶数の場合：
		　どなたでも用途を問わずご使用いただくことができます。PDFなどの文書への埋め
		込みも自由です。
		　このフォントを改造される場合、オリジナルと紛らわしくないようフォント名を
		変更することで許可します。改造したものを再配布することも自由です。

	▼マイナーバージョンが奇数の場合：
		　使用は無料ですが、改造、再配布は禁止します。但し、PDFなどドキュメントへの
		埋め込みはこの限りではありません。
